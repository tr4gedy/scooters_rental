Техническое задание: Аренда самокатов

## 1. Описание проекта
Данный проект представляет собой разработку программной системы для аренды самокатов. 
Система позволяет просматривать доступные самокаты, арендовать их, начинать и завершать поездку, а также просматривать историю поездок.

Тип работы: учебная практика

---

## 2. Цель и задачи проекта
**Цель:** разработка системы аренды самокатов.

**Задачи:**
- анализ предметной области
- разработка архитектуры системы
- реализация программного решения
- тестирование и оценка эффективности

**Область применения:** пользовательское

---

## 3. Функциональность
- мониторинг доступных самокатов
- начать поездку
- закончить поездку
- история поездок

**Ограничения:**
- тестовый режим эксплуатации

---

## 4. Архитектура проекта
Тип архитектуры: клиент-сервер

**Технологии:**
- С№
- Visual Studio Community
- mssql

**Структура проекта:**
- `prct/` — исходный код
- `prct/infrastructure` — серверная часть
- `prct/view` — клиентская часть
- `prct.tests/` — тесты

---

## 5. Установка и запуск

**Требования:**
- .NET 8
- WPF
- Docker (опционально)

**Установка:**
```bash
git clone https://github.com/tr4gedy/scooters_rental
cd scooters_rental
```

**Запуск:**
```bash
dotnet run
```

---

## 6. Использование
После запуска системы пользователь может:
- просматривать данные
- начинать и заканчивать поездку
- просматривать историю поездок
